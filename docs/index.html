QKart Web Testing Project

A manual QA portfolio project demonstrating test planning, test cases, bug reporting, and test execution for an e-commerce web application.

Project Overview

This project documents the end-to-end manual testing process for the QKart web application, a demo e-commerce platform. It showcases a systematic approach to quality assurance, from initial planning and test case design to defect management and final reporting. The goal is to ensure the application's core functionalities are reliable, user-friendly, and meet quality standards.
